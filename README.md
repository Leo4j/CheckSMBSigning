# CheckSMBSigning
Checks for SMB signing disabled on all hosts in the network

Run as follows:

```
iex(new-object net.webclient).downloadstring('https://raw.githubusercontent.com/Leo4j/CheckSMBSigning/main/CheckSMBSigning.ps1')
```
